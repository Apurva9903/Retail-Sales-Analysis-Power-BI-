# Retail Sales Analysis (Power BI)
A data-driven dashboard analyzing retail performance using a Star Schema model.

## 🚀 Key Features
Top/Bottom Performance: Identified top 5 revenue-generating products (Led by Apple iPhone 14).

Geospatial Insights: Sales tracking across 16+ cities, with Bhopal and Kanpur as lead markets.

Trend Analysis: Monthly sales and profit monitoring to identify seasonal peaks.

Data Modeling: Integrated Customer, Product, and Promotion dimensions with transactional data.

## 🛠️ Tools Used
Power BI: Data visualization & DAX.

Power Query: ETL (Data cleaning and merging).

Excel: Source dataset ("Store+Data (1).xlsx").
